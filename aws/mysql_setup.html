MySQL RDS Setup Guide

  • Go to AWS Management Console
  • Navigate to RDS service.

  • Click Databases → Create database
  • Choose Standard Create for database creation method
  • Select MySQL as engine type
  • Select latest stable version (e.g., MySQL 8.0.x)

  • DB instance identifier: e.g., MySQL-RDS-Instance
  • Master username: e.g., admin
  • Set a strong master password and confirm it

  • Under DB instance class, select the size. Example: db.t3.micro for testing

  • Storage type: General Purpose (SSD) or Provisioned IOPS (SSD)
  • Allocated storage: e.g., 20 GB
  • Enable storage autoscaling if needed

  • Select your VPC (e.g., MyVPC)
  • Choose Subnet group (public/private as needed)
  • Enable Public access if external connection is required
  • Attach security group: allow port 3306 from EC2 or your IP

  • DB parameter group: default unless custom settings needed
  • Option group: default
  • Enable automatic backups (recommended)

  • Enable Enhanced monitoring if required
  • Enable Auto minor version upgrade
  • Set Maintenance window (optional)

  • Review all settings carefully
  • Click Create database
  • Wait until status becomes Available

  • Go to Databases → select your DB → Connectivity & security → Endpoint
  • Use this endpoint in MySQL Workbench or CLI:
    • mysql -h <endpoint> -P 3306 -u admin -p
  • Ensure security group allows connections from your EC2 or IP
