Introduction to Data Structures

Data structures help in organizing and managing data efficiently for various computational operations.

What are Data Structures?

Data structures are specialized formats for organizing, processing, retrieving, and storing data. They are essential for building efficient algorithms.

Types of Data Structures

  • Arrays - Fixed-size collection of similar data types.
  • Linked List - Elements linked using pointers.
  • Stack - LIFO (Last In, First Out) data structure.
  • Queue - FIFO (First In, First Out) data structure.
  • Hash Table - Key-value pairs for fast lookup.
  • Trees - Hierarchical data structure (Binary Tree, BST, etc.).
  • Graphs - Nodes and edges representing relationships.

Why are Data Structures Important?

Data structures are crucial for:

  • Efficient data access and modification.
  • Reducing complexity in algorithms.
  • Optimizing memory usage.
  • Handling large amounts of data efficiently.

Example: Implementing a Stack in Java

        
import java.util.Stack;

public class StackExample {
public static void main(String[] args) {
    Stack stack = new Stack<>();
    stack.push(10);
    stack.push(20);
    stack.push(30);
    System.out.println("Top Element: " + stack.peek());
    stack.pop();
    System.out.println("After Pop: " + stack);
}
}
